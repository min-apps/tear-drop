오정민(이하 "개발자")은 Teary(천연눈물) 앱(이하 "앱")의 이용자 개인정보 보호를 매우 중요시하며, 「개인정보 보호법」 등 관련 법령을 준수합니다. 본 개인정보처리방침은 앱이 수집하는 정보의 종류, 이용 목적, 보관 방법 등을 안내합니다.

1. 수집하는 정보

본 앱은 회원가입 절차 없이 익명 인증 방식으로 운영되며, 이름, 이메일, 전화번호 등 개인을 식별할 수 있는 정보는 일절 수집하지 않습니다.

본 앱은 개인 식별 정보를 수집하지 않습니다. 모든 데이터는 익명으로 처리됩니다.
항목 설명 수집 목적
익명 인증 UID Firebase Anonymous Auth에 의해 자동 생성되는 고유 식별자 사용자 데이터 구분 및 동기화
눈물 평가 기록 영상 시청 후 사용자가 입력한 눈물 등급 (평점) 앱 핵심 기능 제공
감정 태그 사용자가 선택한 감정 태그 정보 콘텐츠 분류 및 추천
시청 기록 사용자가 시청한 영상 목록 및 시점 시청 이력 관리
영상 북마크 사용자가 저장한 영상 즐겨찾기 정보 북마크 기능 제공

2. 정보의 이용 목적

수집된 정보는 다음의 목적으로만 이용됩니다.

  • 앱 서비스 제공 및 기능 운영 (눈물 평가, 감정 태그, 북마크 등)
  • 사용자 데이터의 기기 간 동기화
  • 앱 이용 패턴 분석을 통한 서비스 개선 (익명 통계)

3. 정보의 저장 및 보관

모든 데이터는 Firebase(Google Cloud)에 저장되며, 서버 위치는 asia-northeast3(서울) 리전입니다.

  • 데이터 전송 시 HTTPS/TLS 암호화 적용
  • Firebase Security Rules에 의한 접근 제어
  • 사용자 본인의 데이터에만 접근 가능

4. 정보의 보유 및 파기

  • 앱 사용 기간 동안 데이터를 보유합니다.
  • 앱 삭제 시 기기 내 로컬 데이터는 자동 삭제됩니다.
  • 서버에 저장된 익명 데이터의 삭제를 원하시는 경우, 아래 연락처로 요청해주시면 처리해드립니다.

5. 제3자 서비스

본 앱은 다음의 제3자 서비스를 이용합니다.

서비스 제공업체 용도
Firebase Authentication Google LLC 익명 인증
Cloud Firestore Google LLC 데이터 저장
Firebase Analytics Google LLC 익명 사용 통계 분석
YouTube Google LLC 영상 콘텐츠 임베딩

각 서비스의 개인정보처리방침은 해당 업체의 정책을 따릅니다.

  • Google 개인정보처리방침
  • Firebase 개인정보처리방침
  • YouTube 서비스 약관

6. 이용자의 권리

  • 이용자는 언제든지 자신의 데이터 삭제를 요청할 수 있습니다.
  • 앱 삭제를 통해 로컬 데이터를 즉시 삭제할 수 있습니다.
  • 서버 데이터 삭제 요청은 아래 연락처로 문의해주세요.
